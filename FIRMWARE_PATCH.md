# Прошивка ровера — XIAO_ESP32S3_Motors_Laser_Servo_NEW_PCNT

## Текущее состояние

Прошивка включает энкодеры (PCNT) и передаёт `rpmL`/`rpmR` в телеметрии.
OdometryTracker работает в точном режиме из коробки.

## Протокол: телефон → ровер (UDP :4210)

```
SPD:{};STR:{};FWD:{};LASER:{}\n
```

| Поле  | Диапазон   | Как использует прошивка                        |
|-------|------------|------------------------------------------------|
| SPD   | -100..100  | **Игнорируется**                               |
| STR   | -100..100  | `map(-100,100, 40,140)` → серво руления        |
| FWD   | -100..100  | `abs(FWD)` → мощность мотора, `sign(FWD)` → направление |
| LASER | 0 / 1      | GPIO17 вкл/выкл                                |

### Маппинг руления (STR → серво)

```
STR: -100  →  40°  (полностью влево)
STR:    0  →  90°  (прямо)
STR: +100  → 140°  (полностью вправо)
```

**Код:**
```cpp
steerTarget = map(str, -100, 100, 40, 140);
```

## Протокол: телефон → Xiao Sense (UDP :4210)

```
PAN:{};TILT:{}\n
```

| Поле | Диапазон | Маппинг прошивки                              |
|------|----------|-----------------------------------------------|
| PAN  | -100..100 | `map(-100,100, 45,135)` → серво pan           |
| TILT | -100..100 | `map(-100,100, 20,160)` → серво tilt          |

## Телеметрия: ровер → телефон (UDP :4211)

Отправляется каждые 500мс:

```json
{
  "bat": 100,
  "yaw": 0.0,
  "spd": 75,
  "pit": 0.0,
  "rol": 0.0,
  "rssi": -45,
  "rpmL": 120.5,
  "rpmR": 118.3
}
```

| Поле    | Тип   | Описание                              |
|---------|-------|---------------------------------------|
| bat     | int   | Батарея % (захардкожено 100)          |
| yaw     | float | Не реализовано (0.0)                  |
| spd     | int   | Последний `abs(FWD)` из команды       |
| pit,rol | float | Не реализовано (0.0)                  |
| rssi    | int   | WiFi RSSI (dBm)                       |
| rpmL    | float | Обороты левого колеса (PCNT энкодер)  |
| rpmR    | float | Обороты правого колеса (PCNT энкодер) |

## MJPEG стрим: Xiao Sense → телефон (HTTP :81)

Камера OV2640 на XIAO Sense стримит MJPEG по HTTP.
Ось камеры совпадает с лучом лазера.

| Endpoint                          | Описание                         |
|-----------------------------------|----------------------------------|
| `http://192.168.88.23:81/stream`  | MJPEG стрим (multipart)          |
| `http://192.168.88.23/capture`    | Одиночный JPEG кадр              |
| `http://192.168.88.23/status`     | JSON: pan, tilt, heap            |

## Аппаратная конфигурация

### Ровер (192.168.88.24) — rover_esp32s3.ino

| Компонент        | Пины    |
|------------------|---------|
| Мотор IN1/IN2    | 6, 7    |
| PWM L/R          | 4, 5    |
| Энкодер L (A/B)  | 35, 36  |
| Энкодер R (A/B)  | 38, 37  |
| Серво руля       | 13      |
| Серво Pan        | 16      |
| Серво Tilt       | 11      |
| Лазер            | 17      |

**Энкодеры:** 440 пульсов/оборот, квадратурный счёт через PCNT (аппаратный).
Период замера: 200 мс.

### Xiao Sense турель (192.168.88.23) — sketch_mar16a.ino

| Компонент       | Пин                       |
|-----------------|---------------------------|
| Серво Tilt      | D2 (GPIO 2)               |
| Серво Pan       | D3 (GPIO 4)               |
| Камера OV2640   | Встроенная (expansion board) |

**Серво:** 50Hz PWM, 500-2400μs. Полный ход: 0°..180° (обе оси).
